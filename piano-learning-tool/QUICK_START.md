# Piano Master Pro - Quick Start Guide

## 🎹 Play Your First Note in 30 Seconds

1. **Open the app** in your browser
2. **Press any letter key** to play a note:
   - Press **A** to play C
   - Press **S** to play D
   - Press **W** to play C#
3. **Click Settings** to adjust volume if needed

Done! 🎵

---

## ⌨️ Keyboard Layout

```
WHITE KEYS:    A    S    D    F    G    H    J    K
               C    D    E    F    G    A    B    C

BLACK KEYS:    W         E              T    Y    U
              C#       D#             F#   G#  A#
```

## 🆘 Quick Fixes

**No sound?**
- Check system volume
- Click app to initialize audio
- Check Settings volume slider

**Keys don't work?**
- Click anywhere first
- Try keyboard keys A, S, D, F, G, H, J
- Check console for errors

**Visual keys won't play?**
- Make sure audio is initialized
- Check volume settings
- Try keyboard keys instead
