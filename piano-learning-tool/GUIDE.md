## Getting Started

### First Launch
1. Open the application in your web browser
2. You'll see a loading screen - wait for it to complete
3. The dashboard will display with your progress and statistics
4. Start with the **Lessons** section to learn piano basics

### Audio Setup
The application uses synthesized piano sounds. Audio will automatically initialize when you:
- Click anywhere on the screen
- Press your first keyboard key
- Click on any visual piano key

## Keyboard Shortcuts

### Playing Notes (Octave 4 - Middle Octave)
The keyboard is arranged like a piano keyboard for intuitive playing:

#### White Keys
| Key | Note |
|-----|------|
| A | C |
| S | D |
| D | E |
| F | F |
| G | G |
| H | A |
| J | B |
| K | C (next octave) |

#### Black Keys (Sharps)
| Key | Note |
|-----|------|
| W | C# |
| E | D# |
| T | F# |
| Y | G# |
| U | A# |

### Octave Navigation
| Key | Action |
|-----|--------|
| Z | Lower octave (goes down) |
| X | Raise octave (goes up) |

### Additional Controls
| Key | Action |
|-----|--------|
| Space | Toggle metronome on/off |
| R | Start/stop recording (in Practice mode) |

### Mouse Controls
- **Click on piano keys** to play notes directly
- **Touch on mobile devices** also works
- Hover effects show which key will play

## Tips for Learning

### 1. **Start with Basics**
- Master the white keys before moving to black keys
- Learn Middle C first - it's the center reference point
- Practice the C Major scale regularly

### 2. **Use the Metronome**
- Practice scales with the metronome for rhythm
- Start at slower tempos (60-80 BPM)
- Gradually increase as you improve

### 3. **Record and Review**
- Use the recording feature to track progress
- Review your recordings to identify improvement areas
- Record yourself daily to build consistency

### 4. **Practice Regularly**
- Consistent practice beats occasional long sessions
- Even 15-30 minutes daily is better than 3 hours once a week
- The app tracks your streak - keep it going!

### 5. **Enable Helpful Features**
- Keep "Show Labels" on while learning
- Keep "Show Shortcuts" visible until you memorize keys
- Disable them once comfortable for cleaner aesthetics

### 6. **Follow Lessons in Order**
- Each lesson builds on the previous one
- Objectives are designed progressively
- Don't skip ahead without mastery

### 7. **Combine Keyboard and Mouse**
- Use keyboard for scales and drills (faster)
- Use visual mouse clicks for visualization
- Learn both input methods

## Troubleshooting

### Keyboard Keys Not Working
**Solution:**
- First, click anywhere on the app to initialize audio
- If still not working, check browser console for errors
- Ensure your keyboard isn't locked (check Caps Lock)

### No Sound
**Possible causes:**
- System volume is muted
- Browser volume is low
- Audio hasn't been initialized yet (click the app first)
- Your browser might not support Web Audio API

**Solutions:**
1. Check system volume is not muted
2. Click on app, check Settings volume slider
3. Try clicking on a visual key first
4. Try a different browser (Chrome, Firefox, Safari recommended)

### Can't See Keyboard Shortcuts
**Solution:**
- Go to Settings page
- Check the "Show Shortcuts" option
- Keys with shortcuts will display their keyboard mapping

### Progress Not Saving
**Solution:**
- The app automatically saves to your browser's local storage
- Clear browser cache might delete saved data
- Use "Export Data" in settings to backup your progress
- Use "Import Data" to restore from backup

### Sounds Are Distorted
**Solution:**
- Lower the volume slider in Settings
- Adjust system volume
- The synthesized sounds can clip if all frequencies overlap
- Disable "Animations" for better performance

### Metronome Not Working
**Solution:**
- Check that audio is initialized first
- Try toggling off/on with spacebar
- Verify system volume isn't muted
- Check if "Show Labels" or other UI elements are covering the button

## Frequently Asked Questions

**Q: How do I know if I'm doing well?**
A: Check the Dashboard for accuracy percentage and achievements. The app tracks all your progress.

**Q: Can I export my progress?**
A: Yes! Go to Settings and click "Export Data" to download a backup of all your progress, then use "Import Data" to restore it.

**Q: What if I want to start over?**
A: Go to Settings and click "Reset Progress". Warning: This cannot be undone!

**Q: How do I use a MIDI keyboard?**
A: The app supports MIDI devices. Go to Settings to select your connected MIDI input device.

**Q: Can I play songs by ear after learning?**
A: Yes! The Practice section has a "Free Play" mode where you can play anything. Record yourself to track improvement.

**Q: How often should I practice?**
A: For best results, aim for daily practice sessions of 20-30 minutes. The app tracks your practice streak!

## Additional Resources

- **The Piano Keys Layout**: The white keys spell out the note names: A, B, C, D, E, F, G
- **Octaves**: Each octave is a span of 12 notes. Middle C (C4) is the center reference
- **Black Keys (Sharps)**: Located between white keys, played with W, E, T, Y, U
- **Chords**: Combinations of notes played together (see Lessons #4 and #7)
- **Scales**: Sequential notes played in order (C Major scale is C, D, E, F, G, A, B, C)
